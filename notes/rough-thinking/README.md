# Rough Thinking

Use this folder for early reasoning that is not yet formal enough for `docs/`.

Suitable content:

- first-principles decomposition
- tradeoff sketches
- rough technical options
- uncertain stakeholder assumptions
- proposal-framing attempts
- "what would make this budget-fit?" notes

Recommended template: `templates/rough-thinking-note.md`.

Promotion rule: if a rough note changes future action, promote it into `docs/`, `data-contracts/`, `decision-log/`, or `experiments/`.
