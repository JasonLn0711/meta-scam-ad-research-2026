# Meeting Notes

Use this folder for notes from conversations with professors, anti-fraud domain experts, police/CIB stakeholders, legal/policy reviewers, and engineering teammates.

Recommended template: `templates/meeting-note.md`.

Naming:

```text
YYYY-MM-DD-stakeholder-topic.md
```

Keep notes factual and separable:

- what was said
- what was decided
- what is still uncertain
- what should be promoted into formal docs or decision records
