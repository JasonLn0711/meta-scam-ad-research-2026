# Scripts

Future helper scripts may live here.

This is intentionally code-light. Do not add scripts until a documented experiment or data contract needs them.

Rules:

- Python 3.
- Standard library first.
- No automated Meta collection unless approved in `governance/data-governance.md`.
- No credentials in code.
- Scripts should print clear usage and fail with clear messages.

Possible future scripts:

- schema validator for evidence records
- annotation consistency checker
- redaction checker
- experiment summary generator
