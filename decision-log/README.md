# Decision Log

Architecture, scope, data, and research-direction decisions live here over time.

Use this folder for durable decisions that future collaborators should understand without reading the entire repo history.

## Naming

```text
0001-short-decision-title.md
0002-short-decision-title.md
```

Use `templates/decision-record.md`.

## What Belongs Here

- scope narrowing decisions
- architecture decisions
- data access posture changes
- evaluation priority changes
- budget-fit path changes
- decisions to defer or reject a tempting workstream

## What Does Not Belong Here

- meeting raw notes
- experiment run details
- TODO lists
- private scratch thinking
