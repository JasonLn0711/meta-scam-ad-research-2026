# Modality Studies

Use this folder to compare evidence modalities.

Examples:

- text-only vs OCR-only
- image-only vs image + caption
- image + caption + OCR
- carousel image-set handling
- landing-page augmentation
- short-video keyframe + ASR + OCR pilot

Each study must state what narrowing decision it informs.

Recommended template: `templates/experiment-log.md`.
