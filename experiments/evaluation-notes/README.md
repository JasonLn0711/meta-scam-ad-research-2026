# Evaluation Notes

Use this folder for evaluation design, reviewer feedback, error analysis, and metric interpretation.

Good notes include:

- false-positive themes
- false-negative themes
- reviewer disagreement
- evidence completeness analysis
- high/medium/low queue usefulness
- cost per item
- recommendation changes caused by results

These notes should feed `docs/06-evaluation-framework.md` and `docs/10-decision-memo.md`.
