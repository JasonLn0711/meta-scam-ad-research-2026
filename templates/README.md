# Templates

Reusable Markdown templates for this research repo.

Use templates to keep collaboration consistent without turning the project into a rigid database.

Available templates:

```text
meeting-note.md
rough-thinking-note.md
stakeholder-interview.md
scoping-memo.md
experiment-log.md
decision-record.md
data-access-decision.md
```

Prefer copying a template into the relevant folder, then editing it down to what is actually useful.
